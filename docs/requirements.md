# 需求规格说明书

## 项目名称

家庭趣味点菜（暂定名）

## 项目目标

为家庭用户提供一款运行在安卓手机上的趣味点菜工具，帮助家庭成员协作决定每日餐食、管理冰箱食材。

## 用户画像

- 家庭成员（2-6人）
- 安卓手机用户
- 无需技术背景

## 核心功能

### 1. 首页
- **随机点菜**：醒目的"今天吃什么 🐾"大按钮，点击随机显示菜品
- **搜索菜品**：搜索框支持按名称搜索菜品
- **分类筛选**：胶囊标签（全部/荤菜/素菜/汤品/主食）
- **猫咪推荐**：推荐卡片展示随机菜品，支持"换一个"和"就它了！"
- **快捷入口**：三合一卡片（今日菜单、冰箱管理、全部菜品）

### 2. 全部菜品
- **分类浏览**：水平滚动的分类标签
- **网格展示**：两列网格，每格含图片、菜名、分类标签
- **菜品详情**：点击弹出详情弹窗（名称、分类、图片、外部链接）
- **自定义添加**：悬浮"+"按钮，添加新菜品（名称、分类、外部链接）
- **预设数据**：内置50道家常菜

### 3. 今日菜单
- **当日菜单**：显示今天已点的菜品列表
- **长按删除**：长按菜品卡片可删除
- **日期选择**：日历功能翻阅历史点菜记录
- **缺省状态**：无菜品时显示引导插画和"去选菜"按钮

### 4. 冰箱管理
- **食材录入**：添加食材名称、购买日期、保质期
- **食材列表**：展示现有食材，按保质期排序
- **临近过期提醒**：保质期临近的食材高亮显示
- **删除食材**：支持移除已用完的食材
- **缺省状态**：空冰箱时显示猫咪插画

### 5. 我的（家庭组）
- **用户信息**：头像（emoji）+ 昵称
- **创建家庭组**：输入组名 → 生成6位共享码
- **加入家庭组**：输入共享码 → 加入已有家庭组
- **成员展示**：显示家庭组成员头像列表
- **设置**：清除缓存、关于小程序

## 非功能需求

- **无登录注册**：设备ID + 昵称即用
- **实时同步**：同家庭组成员操作实时可见
- **离线可用**：PWA 支持离线访问
- **响应速度**：操作响应 < 500ms
- **触控友好**：按钮最小可触区域 44×44px

## 页面流程

```
首页 ──→ 全部菜品 ──→ 菜品详情
 │          │
 │          └──→ 添加菜品表单
 │
 ├──→ 今日菜单 ──→ 日历历史
 │
 ├──→ 冰箱管理 ──→ 添加食材
 │
 └──→ 我的 ──→ 创建/加入家庭组
```
